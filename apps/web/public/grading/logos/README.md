# Logos des sociétés de gradation

## Fichiers requis

Place les logos PNG dans ce dossier avec les noms suivants :

- `psa.png` - Logo PSA (Professional Sports Authenticator)
- `bgs.png` - Logo BGS (Beckett Grading Services)
- `cgc.png` - Logo CGC (Certified Guaranty Company)
- `pca.png` - Logo PCA (Professional Card Authenticator)
- `collectaura.png` - Logo CollectAura
- `other.png` - Logo générique pour "Other"

## Spécifications recommandées

- **Format** : PNG avec fond transparent
- **Dimensions** : Hauteur ~100-150px (la largeur s'adaptera automatiquement)
- **Couleur** : Le logo sera rendu en blanc via un filtre CSS, donc la couleur d'origine n'a pas d'importance

## Où trouver les logos

Tu peux :

1. Chercher "PSA logo PNG" sur Google Images (filtrer par licence libre de droits)
2. Utiliser les logos officiels des sites web des sociétés
3. Créer des reproductions simplifiées en respectant leur style

## Notes

Les logos seront affichés en blanc sur l'étiquette colorée en haut des cartes gradées.
